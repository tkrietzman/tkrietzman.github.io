Realistic Text

With Changeable Focal Center

You may click here and type to edit the contents of this text. Move the mouse up and down to change the focal center. This silly effect is accomplished using mostly CSS. This particular version uses quite a bit of JS, though. The trick is overlaying three or more slightly transparent objects and using slightly different rotations for each. In this case, the variation between each is 0.3 degrees. Made by Lucas Bebber. You can look at my CodePen profile (@lbebber) to see other stuff I made.
